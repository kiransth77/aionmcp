# Change Log

All notable changes to the "aionmcp-extension" will be documented in this file.

## [0.1.0] - 2025-10-26

### Added
- Initial release of AionMCP VS Code Extension
- Server lifecycle management (start/stop/restart)
- Tool discovery and exploration tree view
- Interactive tool execution with parameter forms
- Agent monitoring and status tracking
- Real-time server logging and output
- Dashboard with server metrics and statistics
- API specification import functionality
- Bundled AionMCP binary for easy deployment
- Cross-platform support (Windows, macOS, Linux)

### Features
- **Server Management**: One-click server control
- **Tool Explorer**: Tree view with categorized tools
- **Tool Execution**: Interactive execution with history
- **Agent Monitoring**: Real-time agent status updates
- **Dashboard**: Comprehensive server overview
- **Import Support**: Drag-and-drop API specs
- **Configuration**: Flexible server and extension settings

### Technical
- TypeScript implementation with latest dependencies
- VS Code API 1.85.0 compatibility
- Bundled cross-platform binaries
- Modern ESLint and TypeScript configuration
- Comprehensive error handling and logging
